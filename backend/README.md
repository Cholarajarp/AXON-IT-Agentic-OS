# Active Backend

This is the canonical AXON backend.

Run backend-only development from the repository root with:

```bash
npm run dev:backend
```

Or from this folder:

```bash
npm run dev
```

This backend owns the active Fastify API, including Mission Control, Product Factory, Database Pipeline, API Forge, Preview QA, Release Command, Trust Ledger, Agentic Mesh, Production Readiness, Production Runtime gates, and artifact evidence packs.

Do not use `legacy/it-agentic-os/backend-src/` as the current product backend. That folder is migration material only.
